# AI Multi-Agent Assistant

A simple **Agentic AI** project built using **LangGraph** and **Groq LLM**. The assistant intelligently routes user queries to specialized agents.

## Features

* 🌤️ Weather Information
* 💱 Currency Conversion
* 🤖 General Question Answering
* 🧠 Intelligent Agent Routing with LangGraph

## Tech Stack

* Python
* LangGraph
* LangChain
* Groq API
* Open-Meteo API
* ExchangeRate API

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```text
GROQ_API_KEY=your_groq_api_key
```

Run the project:

```bash
python app.py
```

## Example Queries

* What's the weather in Chennai?
* Convert 100 USD to INR
* Who invented Python?
* Explain LangGraph.
